# ANEXO I – Guía rápida inicio para docentes (perfil non técnico)
Este anexo está pensado como unha porta de entrada práctica e segura ao uso educativo da intelixencia artificial, dirixida a docentes sen formación técnica específica.

## Que facer o primeiro día que queres usar IA como docente

**Checklist esencial**

- [x] Ter claro para que quero usar a IA (planificar, materiais, adaptar contidos, etc.).
- [x] Comezar cun uso docente, non directamente co alumnado.
- [x] Empregar a IA para xerar borradores, nunca produtos finais sen revisión.
- [x] Revisar sempre o resultado con criterio pedagóxico.
- [x] Probar cun contido que xa coñezo ben.

!!! tip ""
    *O primeiro contacto coa IA debe ser tranquilo e controlado.*

---

## Erros habituais ao comezar (e como evitalos)

**Checklist de erros comúns**

- [x] Pensar que a IA sempre ten razón.
- [x] Copiar e pegar resultados sen revisar.
- [x] Pedirlle demasiadas cousas á vez.
- [x] Usala sen ter claro o obxectivo educativo.
- [x] Compararse con outros docentes máis avanzados.

!!! tip ""
    *Usar IA con criterio implica cuestionar, revisar e adaptar todo o que produce.*

---

## Que non facer nunca como docente

**Checklist de límites claros**

- [x] Non introducir datos persoais do alumnado (nomes, correos, informes, etc.).
- [x] Non subir exames reais, actas ou documentos sensibles.
- [x] Non usar a IA para avaliar automaticamente nin cualificar.
- [x] Non delegar decisións pedagóxicas na IA.
- [x] Non empregar ferramentas non autorizadas polo centro.

!!! tip ""
    *A protección de datos e a responsabilidade profesional son irrenunciables.*

---

## Como probar a IA sen risco

**Checklist para probas seguras**

- [x] Empregar textos ficticios ou exemplos inventados.
- [x] Usar contidos públicos ou propios.
- [x] Probar primeiro en tarefas non avaliables.
- [x] Anotar que funciona e que non.
- [x] Compartir dúbidas con outros docentes.
